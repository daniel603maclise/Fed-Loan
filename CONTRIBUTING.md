# Contribution Guidelines - Fed Loan

## How to Contribute
We welcome contributions!

- Fork the repository
- Create a new branch
- Make your changes
- Submit a pull request

## Reporting Issues
Use GitHub Issues to report bugs or unexpected behavior.

## Suggesting Features
We encourage feature suggestions that improve usability or performance.

## Code of Conduct
Be respectful, collaborative, and constructive.

## Review Process
All pull requests will be reviewed for quality, structure, and consistency before merging.
